## 🚀 What is Educare BD?

Educare BD is a smart school management platform designed for Bangladeshi schools. It uses **AI-powered face recognition** via CCTV cameras to automate daily attendance, instantly notify parents of absent students, and manage all school operations from a single platform.

No cards. No manual input. Just walk through the gate.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🤖 AI Attendance | Face recognition via CCTV — fully automated |
| 📞 Parent Alerts | SMS + Voice Call for absent students by 12:00 PM |
| 📝 Exam Papers | Auto-generate monthly question papers |
| 💸 Salary Payment | Teacher salary via bKash / mobile banking |
| 📋 Notice Board | Digital notices for students & parents |
| 🍱 Tiffin Request | Parents order tiffin through the app |
| 📚 Book List | Class-wise book lists available in-app |
| 🏖️ Leave Application | Digital leave requests |

## 👥 Who Is It For?

- 🏫 **Schools** with 700–800 students
- 👨‍👩‍👧 **Parents** who want real-time updates
- 👩‍🏫 **Teachers** managing class attendance & exams
- 🧑‍💼 **Admins** overseeing all school operations

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| AI / Face Recognition | Python + DeepFace / InsightFace |
| Backend | Node.js + Express / FastAPI |
| Mobile App | React Native (Android + iOS) |
| Web App | React.js |
| Database | PostgreSQL + Redis |
| Notifications | Twilio (SMS + Voice Call) |
| Payment | bKash API / SSLCOMMERZ |
| Hosting | AWS / DigitalOcean |
